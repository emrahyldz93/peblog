A modern, responsive personal blog and portfolio WordPress theme with dark/light mode, built with hybrid architecture.

== Description ==

Peblog is a feature-rich WordPress theme designed for personal blogs and portfolios. Built with modern hybrid architecture, it offers an exceptional user experience with beautiful design, extensive customization options, and powerful functionality.

Perfect for bloggers, content creators, and portfolio showcases who want a professional-looking website without compromising on performance and usability.

== Features ==

* Dark/Light Mode - Seamless toggle with local storage persistence
* Fully Responsive Design - Perfect on all devices (mobile, tablet, desktop)
* Modern & Clean UI - Beautiful, minimalist interface with smooth animations
* Swiper Sliders - Modern hero slider and latest posts carousel
* Featured Posts Section - Highlight your best content
* Most Read Posts - Popular content showcase with views counter
* Category Filtering - Interactive category-based post filtering
* Author Cards - Beautiful author bio cards on single posts
* Post Views Counter - Track and display post popularity
* Social Media Integration - Support for Instagram, Twitter, LinkedIn, GitHub, Pinterest, Telegram, YouTube, Facebook, Dribbble, Behance
* Social Sharing Buttons - Facebook, Twitter, LinkedIn, WhatsApp, Telegram, Pinterest
* Newsletter Support - Shortcode-based integration with MailChimp, MailPoet, etc.
* Custom Sidebar - Optional left/right sidebar with widget support
* Post Tags - Dynamic colored tags with hover effects
* Customizer Options - Extensive theme customization without coding
* Post Navigation - Previous/Next post navigation
* Archive Pages - Custom templates for category, tag, search, and author archives
* 404 Error Page - Beautiful, helpful error page
* SEO Optimized - Clean code and semantic HTML
* Fast Loading - Optimized performance and minimal dependencies
* Cross-Browser Compatible - Works seamlessly across all modern browsers
* Gutenberg Ready - Full block editor support
* Translation Ready - Fully internationalized with i18n support
* Accessibility - WCAG compliant code structure

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

This theme is compatible with most WordPress plugins.

= How do I change the colors? =

You can customize colors and other theme options in the WordPress Customizer (Appearance > Customize).

= Does this theme support dark mode? =

Yes! The theme includes a built-in dark/light mode toggle.
